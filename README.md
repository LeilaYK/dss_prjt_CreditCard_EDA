## 1. 목적
- 2년 동안(2016년 8월 ~ 2018년 07월) 한국의 약 2000여개 상점의 신용카드 거래내역을 이용하여 데이터 시각화 및 인사이트 발굴

## 2. 설치
Requirements
```
* Python 3.6+
```
Installation
```
pip install numpy
pip install pandas
pip install matplotlib
pip install seaborn
pip install plotly
pip install tqdm
```
## 3. 프로젝트 진행 순서

1. 데이터 확인
2. 전처리
3. 시간에 따른 매출 분석
4. 할부내역 분석
5. 거래취소내역 분석
